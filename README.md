# 💱 API Currency Converter

## 📌 Project Overview

API Currency Converter is a web-based application developed using HTML, CSS, and JavaScript that provides real-time currency conversion using an exchange rate API. Users can select source and target currencies, enter an amount, and instantly view the converted value based on the latest exchange rates.

## 🚀 Features

* Real-time currency conversion
* Live exchange rates using API integration
* Support for multiple international currencies
* Country flag display for selected currencies
* User-friendly interface
* Automatic exchange rate calculation
* Responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Exchange Rate API
* Font Awesome

## 🎮 How to Use

1. Enter the amount to convert.
2. Select the source currency.
3. Select the target currency.
4. Click the "Get Exchange Rate" button.
5. View the converted amount and current exchange rate instantly.
